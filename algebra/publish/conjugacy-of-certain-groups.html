Описание основных групп

Группа Диэдра (Группа симметрий правильного \(n\)-угольника)

Замечание: Группу Диэдра можно задать с помощью определяющего соотношения \(\langle r, s| r^n = e, \,\,\,\, s^{-1} = s, \,\,\,\, srs^{-1} = r^{-1} = r^{n - 1} \rangle\).

Группа Диэдра порождается поворотом на минимальный угол \(r = \frac{2\pi}{n}\) и любой из симметрий \(s\).

При любом \(n\) сопряжены повороты на \(\frac{2\pi k}{n}\) и \(\frac{2\pi (n - k)}{n}\), поэтому при нечетном \(n\) получаем \(\frac{n + 1}2\) классов сопряженности поворотов \(\{e\}, \{\frac{2\pi}n, \frac{2\pi (n - 1)}n\}, \ldots, \{\frac{2\pi (\frac{n - 1}2)}n, \frac{2\pi (\frac{n + 1}2)}n \}\), а при четном поворот на \(\frac{2\pi (\frac{n}2)}n = \pi\) совпадает с поворотом на \(\frac{2\pi (n - \frac{n}2)}n\) и получаем \(\frac{n}2 + 1\) классов сопряженности.

Все симметрии сопряжены при нечетном \(n\). При четном они образуют два класса сопряженности: симметрии, проходящие через середины противоположных сторон и симметрии, проходящие через противоположные вершины.

Получаем при четном \(n\) – \(\frac{n}2 + 3\) классов сопряженности, из которых \(\frac{n}2 - 1\) классов сопярженности поворотов по две штуки, класс сопряженнности \(e\) и класс сопряженности поворота \(\pi\) и \(2\) класса сопряженности симметрий.

При нечетном \(n\) – \(\frac{n + 3}2\) классов сопряженности, из которых \(\frac{n + 1}2 - 1\) классов сопярженности поворотов по две штуки, класс сопряженнности \(e\) и \(1\) класс сопряженности симметрий.

Коммутант группы Диэдра

При нечетном \(n\) – подгруппа поворотов.

Группа кватернионов \(Q_8\)

Замечание: Группу кватернионов можно задать с помощью определяющего соотношения \(\langle \pm 1, i, j, k| 1 = (-1)^2 = i^2 = j^2 = k^2, \,\,\,\, ijk = -1 \rangle\).

В группе кватернионов \(4\) класса сопряженности \(\{\pm 1\}, \{\pm i\}, \{\pm j\}, \{\pm k\}\).
