Ряды Фурье

Определение: Выражение вида \(\frac{a_0}2 + \displaystyle \sum_{n = 1}^{k}(a_n \cos (nx) + b_n\sin(nx))\) – тригонометрический многочлен степени \(k\).

\(\{1, \{\sin(nx)\}_{n = 1}^{\infty}, \{\cos(nx)\}_{n = 1}^{\infty}\}\) – тригонометрическая система.

\(\frac{a_0}2 + \displaystyle \sum_{n = 1}^{\infty}(a_n\cos(nx) + b_n\sin(nx))\) – тригонометрический ряд.

Лемма:

  1. \(\displaystyle\int\limits_{-\pi}^{\pi} 1^2dx = 2\pi\),
  2. \(\displaystyle\int\limits_{-\pi}^{\pi} \sin^2(nx)dx = \pi\),
  3. \(\displaystyle\int\limits_{-\pi}^{\pi} \cos^2(nx)dx = \pi\),
  4. \(\displaystyle\int\limits_{-\pi}^{\pi} \cos(nx)\cos(mx)dx = 0 \,\,\,\, \forall n \ne m\),
  5. \(\displaystyle\int\limits_{-\pi}^{\pi} \sin(nx)\sin(mx)dx = 0 \,\,\,\, \forall n \ne m\),
  6. \(\displaystyle\int\limits_{-\pi}^{\pi} \sin(nx)\cos(mx)dx = 0 \,\,\,\, \forall n \ne m\).

Доказательство:

WOW

  1. \(\,\,\,\,\blacksquare\)
  2. формула понижения степени \(\cos^2(nx) = \frac12(1 + \cos(2nx)) \,\,\,\,\blacksquare\)
  3. см. 2. \(\,\,\,\,\blacksquare\)
  4. косинус суммы в произведение \(\,\,\,\,\blacksquare\)
  5. см. 4. \(\,\,\,\,\blacksquare\)
  6. нечетная по симметричному отрезку \(\,\,\,\,\blacksquare\)

Теорема: \[\frac{a_0}2 + \displaystyle \sum_{n = 1}^{\infty}(a_n\cos(nx) + b_n\sin(nx)) \overset{[-\pi, \pi]}{\to} S(x) \implies \begin{cases} a_n = \frac1{\pi}\displaystyle\int\limits_{-\pi}^{\pi} S(x)\cos(nx)dx\\ b_n = \frac1{\pi}\displaystyle\int\limits_{-\pi}^{\pi} S(x)\sin(nx)dx \end{cases}\]

Доказательство:

Интегралы \(\frac1{\pi}\displaystyle\int\limits_{-\pi}^{\pi} S(x)\cos(nx)dx\) и \(b_n = \frac1{\pi}\displaystyle\int\limits_{-\pi}^{\pi} S(x)\sin(nx)dx\) существуют, так как \(S(x)\) непрерывна.

\[\frac{a_0\cos(kx)}2 + \displaystyle \sum_{n = 1}^{\infty}(a_n\cos(nx) + b_n\sin(nx)) \overset{[-\pi, \pi]}{\to} S(x)\implies\] \[\implies \frac{a_0}2 + \displaystyle \sum_{n = 1}^{\infty}(a_n\cos(nx)\cos(kx) + b_n\sin(nx)\cos(kx)) \overset{[-\pi, \pi]}{\to} S(x)\cos(kx) \implies\]

\[\implies \frac1{\pi}\displaystyle\int\limits_{-\pi}^{\pi} S(x)\cos(nx)dx=\] \[= \frac{a_0}2\frac1{\pi}\displaystyle\int\limits_{-\pi}^{\pi}\cos(kx)dx + \displaystyle \sum_{n = 1}^{\infty}\left(\frac{a_n}{\pi}\displaystyle\int\limits_{-\pi}^{\pi}\cos(nx)\cos(kx)dx + \frac{b_n}{\pi}\displaystyle\int\limits_{-\pi}^{\pi}\sin(nx)\cos(kx)\right) = a_k \,\,\,\,\blacksquare\]

Определение: Пусть \(\exists \displaystyle\int\limits_{-\pi}^{\pi}f(x)dx\) и \(\exists \displaystyle\int\limits_{-\pi}^{\pi}|f(x)|dx\). Тогда \(\exists a_n = \displaystyle\int\limits_{-\pi}^{\pi}f(x)\cos(nx)\) и \(\exists b_n = \displaystyle\int\limits_{-\pi}^{\pi}f(x)\sin(nx)\).

\(a_n\) и \(b_n\) – коэффиценты Фурье функции \(f(x)\).

\(\frac{a_0}2 + \displaystyle \sum_{n = 1}^{\infty}(a_n\cos(nx) + b_n\sin(nx))\) – ряд Фурье функции \(f(x)\).

Фанфакт: \(\exists \displaystyle\int\limits_{-\pi}^{\pi}f^2(x)dx \implies \exists \displaystyle\int\limits_{-\pi}^{\pi}|f(x)|dx\)

Доказательство:

\((|f(x)| - 1)^2 \geqslant 0 \implies |f(x)| \leqslant \frac{1 + f^2(x)}{2}\)

\(\exists \displaystyle\int\limits_{-\pi}^{\pi}\frac{1 + f^2(x)}{2}dx \implies\) по признаку Вейерштрасса \(\exists \displaystyle\int\limits_{-\pi}^{\pi}|f(x)|dx \,\,\,\,\blacksquare\)

Минимальное свойство коэффицентов Фурье

Теорема: (Минимальное свойство коэффицентов Фурье) Пусть \(\exists \displaystyle\int\limits_{-\pi}^{\pi}f(x)dx\) и \(\exists \displaystyle\int\limits_{-\pi}^{\pi}f^2(x)dx\).

\(T_n(x) = \frac{A_0}2 + \displaystyle \sum_{k = 1}^{n}(A_k \cos(kx) + B_k\sin(kx))\) – тригонометрический многочлен степени \(n\).

Пусть \(S_n(x)\) – частичная сумма ряда Фурье функции \(f(x)\), то есть \(S_n(x) = \frac{a_0}2 + \displaystyle \sum_{k = 1}^{n}(a_k \cos(kx) + a_k\sin(kx))\).

Тогда \(\displaystyle\int\limits_{-\pi}^{\pi} (f(x) - S_n(x))^2dx = \displaystyle\min_{T_n: \,\,\,\, \operatorname{deg}T_n \leqslant n}\displaystyle\int\limits_{-\pi}^{\pi} (f(x) - T_n(x))^2dx\)

Доказательство:

\[\displaystyle\int\limits_{-\pi}^{\pi} (f(x) - T_n(x))^2dx = \displaystyle\int\limits_{-\pi}^{\pi} (f(x) - \frac{A_0}2 - \displaystyle \sum_{k = 1}^{n}(A_k \cos(kx) + B_k\sin(kx)))^2dx =\] \[=\displaystyle\int\limits_{-\pi}^{\pi} (f^2(x) + \frac{A^2_0}4 - f(x)A_0 + \ldots)dx \leqslant \ldots \leqslant\] \[\leqslant \displaystyle\int\limits_{-\pi}^{\pi}f^2(x)dx + \frac{\pi}2\left(A_0^2 - a_0^2\right) + \pi \displaystyle \sum_{k = 1}^{n}((A_k - a_k)^2 + (B_k - b_k)^2) - \pi \displaystyle \sum_{k = 1}^{n}(a^2_k + b^2_k) - \frac{\pi}2 a_0^2\]

Минимум последней функции в нужных точках \(\,\,\,\,\blacksquare\)

Неравенство Бесселя

Следствие: (Неравенство Бесселя) В условиях теоремы \(\frac{1}{\pi}\displaystyle\int\limits_{-\pi}^{\pi}f^2(x)dx \geqslant \displaystyle \sum_{k = 1}^{n}(a^2_k + b^2_k) + \frac{a_0^2}2\)

Доказательство: \(\frac{1}{\pi}\displaystyle\int\limits_{-\pi}^{\pi}f^2(x)dx - \displaystyle \sum_{k = 1}^{n}(a^2_k + b^2_k) - \frac{a_0^2}2 \geqslant 0\), так как минимум неотрицательной штуки \(\,\,\,\,\blacksquare\)

Следствие: В условиях теоремы \(a_k \overset{k \to \infty}{\to} 0\) и \(b_k \overset{k \to \infty}{\to} 0\).

Доказательство: Ряд сходится, так как частичные суммы ограничены, а следовательно \(a^2_k + b^2_k \overset{k \to \infty}{\to} 0 \,\,\,\,\blacksquare\)

Лемма: Пусть \(f(x)\) \(2\pi\)-периодическая функция и \(f(x) \in R[-\pi, \pi]\). Тогда \(\forall a \in \mathbb{R}\,\,\,\, \displaystyle\int\limits_{-\pi}^{\pi} f(x)dx = \displaystyle\int\limits_{a-\pi}^{a+\pi} f(x)dx\).

Доказательство: Записываем как сумму интегралов \(\,\,\,\,\blacksquare\)

Определение: Функция \(D_n(t) = \frac12 + \displaystyle \sum_{k = 1}^{n}\cos(kt)\) – ядро Дирихле.

Теорема: (Свойства ядра Дирихле)

  1. \(D_n(x)\) – четная,
  2. \(D_n(x)\) – непрерывная на \(\mathbb{R}\),
  3. \(D_n(x)\) – \(2\pi\)-периодическая,
  4. \(D_n(0) = \frac12 + n\),
  5. \(\frac{1}{\pi}\displaystyle\int\limits_{-\pi}^{\pi} D_n(x)dx = 1\),
  6. \(D_n(x) = \frac{\sin\left(\left(n + \frac12\right)x\right)}{2\sin\left(\frac{x}{2}\right)}\).

Доказательство:

  1. Умножить и разделить на \(\sin\left(\frac{x}2\right) \,\,\,\,\blacksquare\)

Рассуждение

Пусть \(f(x)\) – \(2\pi\)-периодическая функция.

\[S_n(x) = \frac{a_0}2 + \displaystyle \sum_{k = 1}^{n}(a_k \cos (kx) + b_k\sin(kx))=\]

\[=\frac12 \cdot \frac{1}{\pi}\displaystyle\int\limits_{-\pi}^{\pi} f(t)dt + \displaystyle \sum_{k = 1}^{n}\left(\cos(kx)\frac1{\pi}\displaystyle\int\limits_{-\pi}^{\pi}f(t)\cos(kt)dt + \sin(kx)\frac1{\pi}\displaystyle\int\limits_{-\pi}^{\pi}f(t)\sin(kt)dt\right)=\] \[=\frac1{\pi}\displaystyle\int\limits_{-\pi}^{\pi}f(t)\left(\frac12 + \displaystyle \sum_{k = 1}^{n}(\cos(kx)\cos(kt) + \sin(kx)\sin(kt))\right)dt=\] \[=\frac1{\pi}\displaystyle\int\limits_{-\pi}^{\pi}f(t)\left(\frac12 + \displaystyle \sum_{k = 1}^{n}\cos(k(t - x))\right)dt=\] \[= \frac{1}{\pi}\displaystyle\int\limits_{-\pi}^{\pi} f(t)D_n(t - x)dt\]

Сделаем замену \(\tau = t - x\)

Имеем \(S_n(x) = \frac{1}{\pi}\displaystyle\int\limits_{-\pi-x}^{\pi - x} f(\tau + x)D_n(\tau)d\tau=\) \[= \frac{1}{\pi}\displaystyle\int\limits_{-\pi}^{\pi} f(\tau + x)D_n(\tau)d\tau\]

Так как \(D_n(x)\) нечетная функция при замене \(\tau\) на \(-\tau\) получим

\[S_n(x) = \frac1{\pi} \displaystyle\int\limits_{-\pi}^{\pi} f(\tau - x)D_n(\tau)d\tau\]

Сложив последние два равенства, получим:

\[S_n(x) = \frac12 \frac1{\pi}\displaystyle\int\limits_{-\pi}^{\pi}(f(\tau + x) + f(\tau - x))D_n(\tau)d\tau\]

\(f(x + \tau) + f(x - \tau)\) – четная, поэтому

\[S_n(x) = \frac{1}{\pi}\displaystyle\int\limits_0^{\pi}(f(x + \tau) + f(x - \tau))D_n(\tau)d\tau\]

Принцип локализации Римана

Теорема: (принцип локализации Римана)

\(\forall x \in [-\pi, \pi] \,\,\,\, \exists \displaystyle \lim_{n\to \infty}S_n(x) \iff \forall \pi > \delta > 0 \,\,\,\, \exists\displaystyle \lim_{n\to \infty}\displaystyle\int\limits_0^{\delta}(f(x + \tau) + f(x - \tau))D_n(\tau)d\tau\)

Доказательство:

\[\displaystyle\int\limits_{\delta}^{\pi}(f(x + \tau) + f(x - \tau))D_n(\tau)d\tau = \displaystyle\int\limits_{\delta}^{\pi}(f(x + \tau) + f(x - \tau))\frac{\sin(n + \frac12)\tau}{2\sin\left(\frac{\tau}{2}\right)}d\tau =\]

\[= \displaystyle\int\limits_{\delta}^{\pi}(f(x + \tau) + f(x - \tau))\frac{1}{2\sin\left(\frac{\tau}{2}\right)} (\sin(n\tau)\cos\left(\frac12\tau\right) + \cos(n\tau)\sin\left(\frac12\tau\right))d\tau =\]

\[= \displaystyle\int\limits_{\delta}^{\pi}\frac{f(x + \tau) + f(x - \tau)}{2}(\sin(n\tau)\operatorname{ctg}\left(\frac12\tau\right) + \cos(n\tau))d\tau\]

Пусть

\[g(\tau) = \begin{cases} \frac{f(x + \tau) + f(x - \tau)}{2}, \,\,\,\,\tau \in [\delta, \pi]\\ 0, \,\,\,\, \tau \in [0, \delta] \end{cases}\]

\[h(\tau) = \begin{cases} \frac{f(x + \tau) + f(x - \tau)}{2}\operatorname{ctg}\left(\frac12\tau\right), \,\,\,\,\tau \in [\delta, \pi]\\ 0, \,\,\,\, \tau \in [0, \delta] \end{cases}\]

Имеем \(\displaystyle\int\limits_{\delta}^{\pi}(f(x + \tau) + f(x - \tau))D_n(\tau)d\tau = \displaystyle\int\limits_{-\pi}^{\pi}g(\tau)\cos(n\tau) d\tau + \displaystyle\int\limits_{-\pi}^{\pi}h(\tau)\sin(n\tau) d\tau\), а это коэффиценты рядов Фурье функций \(g\) и \(h\), а следовательно по лемме стремятся к нулю при \(n \to \infty\).

\[\displaystyle \lim_{n\to \infty}\left(\displaystyle\int\limits_0^{\delta}(f(x + \tau) + f(x - \tau))D_n(\tau)d\tau + \displaystyle\int\limits_{\delta}^{\pi}(f(x + \tau) + f(x - \tau))D_n(\tau)d\tau\right)=\] \[= \displaystyle \lim_{n\to \infty}\displaystyle\int\limits_0^{\delta}(f(x + \tau) + f(x - \tau))D_n(\tau)d\tau = \displaystyle \lim_{n\to \infty}S_n(x) \,\,\,\,\blacksquare\]

Теорема: (признак Дини) Пусть \(f(x)\) – \(2\pi\)-периодическая функция и \(\exists \displaystyle\int\limits_{-\pi}^{\pi}f(x)dx, \,\,\,\, \exists \displaystyle\int\limits_{-\pi}^{\pi}f^2(x)dx\). Пусть \(x_0 \in [-\pi, \pi]\) и \(\exists \displaystyle \lim_{h\to 0+}f(x_0 + h) = f_{+}(x_0), \,\,\,\,\exists \displaystyle \lim_{h\to 0+}f(x_0 - h) = f_{-}(x_0)\).

Если \(\exists \delta > 0: \,\,\,\, \displaystyle\int\limits_0^{\delta}\frac{|f(x_0 + t) + f(x_0 - t) - f_{+}(x_0) - f_{-}(x_0)|}{t}dt\), то \(\exists \displaystyle \lim_{n\to \infty}S_n(x_0) = \frac{f_{+}(x_0) + f_{-}(x_0)}{2}\).

Доказательство:

\[\exists\displaystyle\int\limits_{0}^{\pi}\frac{|f(x_0 + t) + f(x_0 - t) - f_{+}(x_0) - f_{-}(x_0)|}{2\sin\left(\frac{t}{2}\right)}dt \iff \exists\displaystyle\int\limits_{0}^{\pi}\frac{|f(x_0 + t) + f(x_0 - t) - f_{+}(x_0) - f_{-}(x_0)|}{t}dt\]

\[\left|S_n(x_0) - \frac{f_{+}(x_0) + f_{-}(x_0)}{2}\right|=\] \[= \left|\frac{1}{\pi}\displaystyle\int\limits_{0}^{\pi}(f(x + \tau) + f(x - \tau))D_n(\tau)d\tau - \frac{f_{+}(x_0) + f_{-}(x_0)}{2\pi}\displaystyle\int\limits_{-\pi}^{\pi}D_n(t)dt\right|=\] \[=\left|\frac{1}{\pi}\displaystyle\int\limits_{0}^{\pi}(f(x_0 + t) + f(x_0 - t) - f_{+}(x_0) - f_{-}(x_0))\frac{\sin(n + \frac12)t}{2\sin\left(\frac{t}{2}\right)}dt\right|\]

\[\exists\displaystyle\int\limits_{0}^{\pi}\frac{|f(x_0 + t) + f(x_0 - t) - f_{+}(x_0) - f_{-}(x_0)|}{2\sin\left(\frac{t}{2}\right)}dt \implies\] \[\implies\forall \varepsilon > 0 \,\,\,\, \exists \delta_{\varepsilon} \in (0, \delta): \,\,\,\, \displaystyle\int\limits_{0}^{\delta_{\varepsilon}}\frac{|f(x_0 + t) + f(x_0 - t) - f_{+}(x_0) - f_{-}(x_0)|}{2\sin\left(\frac{t}{2}\right)}dt < \varepsilon\]

Аналогично доказательству в предыдущей теореме \(\displaystyle\int\limits_{\delta}^{\pi}(f(x_0 + t) + f(x_0 - t) - f_{+}(x_0) - f_{-}(x_0))\frac{\sin(n + \frac12)t}{2\sin\left(\frac{t}{2}\right)}dt \overset{n\to \infty}{\to} 0\)

\[\left|S_n(x_0) - \frac{f_{+}(x_0) + f_{-}(x_0)}{2}\right|=\] \[=\displaystyle\int\limits_{0}^{\delta}\left|f(x_0 + t) + f(x_0 - t) - f_{+}(x_0) - f_{-}(x_0)\right|\left|\frac{\sin(n + \frac12)t}{2\sin\left(\frac{t}{2}\right)}\right|dt+\] \[+\displaystyle\int\limits_{\delta}^{\pi}(f(x_0 + t) + f(x_0 - t) - f_{+}(x_0) - f_{-}(x_0))\frac{\sin(n + \frac12)t}{2\sin\left(\frac{t}{2}\right)}dt \leqslant \frac{2\varepsilon}{\pi} \,\,\,\,\blacksquare\]

Следствие: Пусть \(f(x)\) гёльдерова в точке \(x_0\). Тогда ряд Фурье \(f(x)\) в точке \(x_0\) сходится к \(\frac{f_{+}(x_0) - f_{-}(x_0)}2\).

Доказательство:

\(\displaystyle\int\limits_0^{\delta}\frac{|f(x_0 + t) + f(x_0 - t) - f_{+}(x_0) - f_{-}(x_0)|}{t}dt \leqslant \displaystyle\int\limits_0^{\delta}\frac{|f(x_0 + t) - f_{+}(x_0)| + |f(x_0 - t)- f_{-}(x_0)|}{t}dt \leqslant 2\displaystyle\int\limits_0^{\delta}Cx^{\alpha - 1}dx\) – сходится и далее по предыдущей теореме \(\,\,\,\,\blacksquare\)

Утверждение: Пусть \(f(x) \in \mathcal{D}\). Тогда \(f(x)\) гёльдерова.

Доказательство: \(\,\,\,\,\blacksquare\)

Достаточное условие равномерной сходимости ряда Фурье

Теорема: (достаточное условие равномерной сходимости ряда Фурье) Пусть \(f(x) \in C[-\pi, \pi], \,\,\,\, f(-\pi) = f(\pi)\) и \(f(x)\) доопределена до \(2\pi\)-периодической функции на \(\mathbb{R}\). Пусть \(f’(x)\) существует и непрерывна на \([-\pi, \pi]\) всюду кроме конечного числа точек и там где \(f’(x)\) не определена существуют пределы справа и слева. Тогда \(\frac{a_0}2 + \displaystyle \sum_{n = 1}^{\infty}a_n\cos(nx) + b_n\sin(nx) \overset{\mathbb{R}}{\rightrightarrows}\).

Пусть \(f(x)\) \(2\pi\)-периодическая, \(\exists \displaystyle\int\limits_{-\pi}^{\pi}f(x)dx, \,\,\,\, \exists \displaystyle\int\limits_{-\pi}^{\pi}f^2(x)dx\). Если \(f(x) \in \mathcal{D}[-\pi, \pi] \,\,\,\, \exists \displaystyle\int\limits_{-\pi}^{\pi}f’(x)dx, \,\,\,\, \exists \displaystyle\int\limits_{-\pi}^{\pi} \left(f’(x)\right)^2 dx\).

Доказательство:

\(\exists \displaystyle\int\limits_{-\pi}^{\pi} \left(f’(x)\right)^2 dx\), а следовательно по следствию ряд \(\displaystyle \sum_{n = 1}^{\infty}\alpha_n^2 + \beta_n^2\), где \(\alpha_n\) и \(\beta_n\) – коэффиценты Фурье функции \(f’(x)\), сходится.

\[\alpha_n = \frac1{\pi}\displaystyle\int\limits_{-\pi}^{\pi} f’(x)\cos(nx)dx = \frac1{\pi}f(x)\cos(nx)_{\left|_{-\pi}^{\pi}\right.} + \frac1{\pi}n\displaystyle\int\limits_{-\pi}^{\pi} f(x)\sin(nx)dx = a_n\cdot n\]

\[\beta_n = \ldots = -b_n\cdot n\]

\[\left|a_n\cos(nx) - b_n\sin(nx)\right| \leqslant |a_n| + |b_n| = \frac1n\cdot\left(|\alpha_n|+|\beta_n|\right) \leqslant \frac1{n^2}\cdot\left(\alpha_n^2 + 2|\alpha_n||\beta_n| + \beta_n^2\right) \leqslant \frac2{n^2} + \alpha_n^2 + \beta_n^2 \implies\]

\(\implies\) по признаку Вейерштрасса ряд сходится \(\,\,\,\,\blacksquare\)

Достаточное условие дифференцируемости ряда Фурье

Определение: Функция \(f(x)\) гёльдерова, если \(\exists \gamma \in (0, 1)\,\,\,\, \exists C > 0\,\,\,\, \forall \mathbb{R} \,\,\,\, \forall \Delta x > 0 \,\,\,\, |f(x + \Delta x) - f(x)| < C\cdot|\Delta x|^{\gamma}\).

Теорема: (достаточное условие дифференцируемости ряда Фурье) Пусть \(f(x) \in C[-\pi, \pi], \,\,\,\, f(-\pi) = f(\pi)\) и \(f(x)\) доопределена до \(2\pi\)-периодической функции на \(\mathbb{R}\). Пусть \(f’(x)\) гёльдерова.

Тогда \(f’(x) = \displaystyle \sum_{n = 1}^{\infty}\alpha_n \cos(nx) + \beta_n\sin(nx) = \cos(nx) + \beta_n\sin(nx) = \left(\frac{a_0}2 + \displaystyle \sum_{n = 1}^{\infty}a_n\cos(nx) + b_n\sin(nx)\right)’\) по предыдущей теореме.

Доказательство: ? почему гёльдерова соответствует условиям теоремы?

Производная получается без равномерной сходимости формально.

Интегрирование ряда Фурье

Теорема: Пусть \(\exists \displaystyle\int\limits_{-\pi}^{\pi}f(x)dx, \,\,\,\, \exists \displaystyle\int\limits_{-\pi}^{\pi}f^2(x)dx\). \(F(x) = \displaystyle\int\limits_0^x \left(f(t) - \frac{a_0}2\right)dt = \frac{A_0}2 + \displaystyle \sum_{n = 1}^{\infty}-\frac{b_n}n \cos(nx) + \frac{a_n}n \sin(nx)\) и этот ряд равномерно сходится.

Доказательство:

Сходится равномерно по Вейерштрассу.

Запишем коэффиценты Фурье для функции \(F(x)\):

\[A_n = \frac1{\pi} \displaystyle\int\limits_{-\pi}^{\pi}\left(\displaystyle\int\limits_0^x \left(f(t) - \frac{a_0}2\right)dt\right)\cos(nx) dx = \frac1{\pi n} \displaystyle\int\limits_{-\pi}^{\pi}\left(\displaystyle\int\limits_0^x \left(f(t) - \frac{a_0}2\right)dt\right)d\sin(nx) =\] \[= -\frac1{\pi n} \displaystyle\int\limits_{-\pi}^{\pi}f(x)\sin(nx)dx = -\frac1n b_n\]

\[B_n = \frac1n a_n\]

Подставим \(0\): \(0 = \frac{A_0}2 - \displaystyle \sum_{n = 1}^{\infty}\frac{b_n}n \,\,\,\,\blacksquare\)

Пример (обязательный)

\(f(x) = |x|\) на \([-\pi, \pi]\).

Получить \(\displaystyle \sum_{n = 1}^{\infty}\frac1{n^2} = \frac{\pi^2}6\).
