Признаки Дирихле и Абеля сходимости рядов

Признак Дирихле

Теорема: (признак Дирихле) Рассмотрим ряд \(\displaystyle \sum_{n = 1}^{\infty}a_nb_n\). Пусть

  1. \(\exists C > 0: \,\,\,\, \forall N \in \mathbb{N} \,\,\,\, \left|\displaystyle \sum_{n = 1}^{N}b_n\right| \leqslant C\)
  2. \(\{a_n\}_{n = 1}^{\infty}\) монотонна и \(\displaystyle \lim_{n\to \infty}a_n = 0\)

Тогда \(\displaystyle \sum_{n = 1}^{\infty}a_nb_n\) сходится.

Доказательство:

\[\displaystyle \sum_{n = k + 1}^{N}a_nb_n = \displaystyle \sum_{n = k + 1}^{N}a_n(B_n - B_{n - 1})=\] \[= a_{k + 1} B_{k + 1} - a_{k + 1} B_k + a_{k + 2} B_{k + 2} - a_{k + 2} B_{k + 1} + \ldots + a_N B_N - a_N B_{N - 1}=\] \[= B_{k + 1} (a_{k + 1} - a_{k + 2}) + B_{k + 2}(a_{k + 2} - a_{k + 3}) + \ldots + B_N (a_N - a_{N + 1}) - a_{k + 1} B_k=\] \[= \displaystyle \sum_{n = k + 1}^{N}B_n(a_n - a_{n + 1}) - a_{k + 1} B_k\]

\[\left|\displaystyle \sum_{n = k + 1}^{N}a_nb_n\right| = \left|\displaystyle \sum_{n = k + 1}^{N}B_n(a_n - a_{n + 1}) - a_{k + 1} B_k\right| \leqslant \left|\displaystyle \sum_{n = k + 1}^{N}B_n(a_n - a_{n + 1})\right| + \left|a_{k + 1} B_k\right| \leqslant\] \[\leqslant \left|\displaystyle \sum_{n = k + 1}^{N}C(a_n - a_{n + 1})\right| + \left|a_{k + 1}C\right| =\] \[=\left|C(a_{k + 1} - a_N)\right| + \left|a_{k + 1} С\right| \leqslant C(2|a_{k + 1}| + |a_N|) \leqslant 3C \varepsilon \,\,\,\,\blacksquare\]

Признаки Абеля и Лейбница

Следствие: (признак Абеля) Рассмотрим ряд \(\displaystyle \sum_{n = 1}^{\infty}a_nb_n\). Пусть

  1. Ряд \(\displaystyle \sum_{n = 1}^{\infty}b_n\) сходится.
  2. Последовательность \(\{a_n\}_{n = 1}^{\infty}\) ограниченна и монотонна

Тогда \(\displaystyle \sum_{n = 1}^{\infty}a_nb_n\) сходится.

Доказательство:

Последовательность \(\{a_n\}_{n = 1}^{\infty}\) ограниченна и монотонна, а значит \(\exists \displaystyle \lim_{n\to \infty}a_n = a\) следовательно \(\displaystyle \lim_{n\to \infty}(a_n - a) = 0\).

\[\displaystyle \sum_{n = 1}^{\infty}a_nb_n = \displaystyle \sum_{n = 1}^{\infty}(\left[a_n - a\right] + a)b_n = \displaystyle \sum_{n = 1}^{\infty}(a_n - a)b_n + a\displaystyle \sum_{n = 1}^{\infty}b_n\]

Ряд \(\displaystyle \sum_{n = 1}^{\infty}(a_n - a)b_n\) сходится по признаку Дирихле, а ряд \(a\displaystyle \sum_{n = 1}^{\infty}b_n\) сходится по условию \(\,\,\,\,\blacksquare\)

Следствие: (признак Лейбница) Если \(\displaystyle \lim_{n\to \infty}a_n = 0\) и \(a_n\) монотонна, то ряд \(\displaystyle \sum_{n = 1}^{\infty}(-1)^{n + 1}a_n\) сходится.

Доказательство: \(\,\,\,\,\blacksquare\)

Функциональные последовательности и ряды

Определения

Определение: Последовательность \(\{f_n: \,\,\,\, X \to \mathbb{R}\}_{n = 1}^{\infty}\) 1 - функциональная последовательность. 2

Определение: Функциональная последовательность \(\{f_n: \,\,\,\, X \to \mathbb{R}\}_{n = 1}^{\infty}\) ограничена на множестве \(X\), если \(\exists C > 0: \,\,\,\, \forall n \in \mathbb{N} \,\,\,\, \forall x \in X \,\,\,\, |f_n(x)| \leqslant C\).

Определение: Функциональная последовательность \(\{f_n: \,\,\,\, X \to \mathbb{R}\}_{n = 1}^{\infty}\) сходится, если \(\forall x_0 \in X\) последовательность \(\{f_n(x_0)\}_{n = 1}^{\infty}\) сходится.

Если функциональная последовательность сходится, то функция \(f: X \to \mathbb{R}, \,\,\,\, f: x_0 \mapsto \displaystyle \lim_{n\to \infty}f_n(x_0)\) - предел (функциональной) последовательности.

Обозначается: \(f_n \xrightarrow[X]{} f\)

В кванторах: \(\forall x_0 \in X \,\,\,\, \forall \varepsilon > 0 \,\,\,\, \exists N_{x_0} \in \mathbb{N}: \,\,\,\, \forall n > N_{x_0} \,\,\,\, |f_n(x_0) - f(x)| < \varepsilon\)

Определение: Функциональная последовательность \(\{f_n: \,\,\,\, X \to \mathbb{R}\}_{n = 1}^{\infty}\) равномерно сходится к функции \(f: X \to \mathbb{R}\), если \(\forall \varepsilon > 0 \,\,\,\, \exists N \in \mathbb{N}: \,\,\,\, \forall n > N \,\,\,\, \forall x \in X \,\,\,\, |f_n(x) - f(x)| < \varepsilon\).

Обозначается: \(f_n \underset{X}{\rightrightarrows} f\)

Определение: Ряд \(\displaystyle \sum_{n = 1}^{\infty}f_n\) на множестве \(X\) - это \(\{\displaystyle \sum_{n = 1}^{\infty}f_n(x_0)| \,\,\,\, x_0 \in X\}\).

\(s_N(x_0) = \displaystyle \sum_{n = 1}^{N}f_n(x_0), \,\,\,\, x_0 \in X\) - частичная сумма ряда.

Ряд \(\displaystyle \sum_{n = 1}^{\infty}f_n\) на множестве \(X\) сходится, если сходится функциональная последовательность \(\{s_N(x)\}_{n = 1}^{\infty}\) его частичных сумм.

Критерии равномерной сходимости функциональной последовательности

Лемма: (критерий равномерной сходимости последовательности) \(f_n \underset{X}{\rightrightarrows} f \iff \displaystyle \sup_{X}|f_n(x) - f(x)| \underset{n \to \infty}{\rightarrow} 0\) 3

Доказательство: определение \(\,\,\,\,\blacksquare\)

Теорема: (критерий Коши равномерной сходимости функциональной последовательности) \(f_n \underset{X}{\rightrightarrows} f \iff \forall \varepsilon > 0 \,\,\,\, \exists N \in \mathbb{N}: \,\,\,\,\forall n, m > N \,\,\,\, \forall x \in X \,\,\,\, |f_n(x) - f_m(x)| < \varepsilon\).

Доказательство: \[\Longrightarrow\] Как обычно прибавить и вычесть \(f(x) \,\,\,\,\blacksquare\) \[\Longleftarrow\] Чего-то замутить 4 \(\,\,\,\,\blacksquare\)
