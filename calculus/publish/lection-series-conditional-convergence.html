Условно сходящиеся ряды

Определение: Если \(\displaystyle \sum_{n = 1}^{\infty}a_n\) сходится, а \(\displaystyle \sum_{n = 1}^{\infty}|a_n|\) расходится, то \(\displaystyle \sum_{n = 1}^{\infty}a_n\) - условно сходящийся.

Утверждение: Рассмотрим \(\displaystyle \sum_{n = 1}^{\infty}a_n\). Обозначим \(a_n^{+} = \begin{cases}a_n,\,\,\,\,a_n \geqslant 0 \\ 0, \,\,\,\, a_n < 0\end{cases}\), \(a_n^{-} = \begin{cases}0,\,\,\,\,a_n \geqslant 0 \\ a_n, \,\,\,\, a_n < 0\end{cases}\).

Если \(\displaystyle \sum_{n = 1}^{\infty}a_n\) сходится условно, то ряды \(\displaystyle \sum_{n = 1}^{\infty}a_n^{+}\) и \(\displaystyle \sum_{n = 1}^{\infty}a_n^{-}\) расходятся.

Доказательство: От противного. Если \(\displaystyle \sum_{n = 1}^{\infty}a_n^{+}\) и \(\displaystyle \sum_{n = 1}^{\infty}a_n^{-}\) сходятся, то \(\displaystyle \sum_{n = 1}^{\infty}|a_n| = \displaystyle \sum_{n = 1}^{\infty}a_n^{+} - \displaystyle \sum_{n = 1}^{\infty}a_n^{-}\) - сходится.

Пусть теперь \(\displaystyle \sum_{n = 1}^{\infty}a_n^{+}\) сходится, а \(\displaystyle \sum_{n = 1}^{\infty}a_n^{-}\) расходится, тогда \(\displaystyle \sum_{n = 1}^{\infty}a_n^{-} = \displaystyle \sum_{n = 1}^{\infty}a_n - \displaystyle \sum_{n = 1}^{\infty}a_n^{+}\) сходится \(\,\,\,\,\blacksquare\)

Теорема: (Римана об условно сходящихся рядах) Если ряд \(\displaystyle \sum_{n = 1}^{\infty}a_n\) сходится условно, то \(\forall a \in \mathbb{R} \,\,\,\, \exists \sigma: \mathbb{N} \to \mathbb{N}\,\,\,\,\), такая, что \(\displaystyle \sum_{n = 1}^{\infty}a_{\sigma(n)} = A\). Далее, \(\exists \sigma\), при которой ряд \(\displaystyle \sum_{n = 1}^{\infty}a_{\sigma(n)}\) расходится на любой вкус и цвет 1.

Доказательство: \(\displaystyle \sum_{n = 1}^{\infty}a_n\) сходится условно \(\implies\) \(\displaystyle \sum_{n = 1}^{\infty}a_n^{+}\) возрастает к \(+\infty\), \(\displaystyle \sum_{n = 1}^{\infty}a_n^{-}\) убывает к \(-\infty\).

  1. Расходится к \(\pm \infty\), расходится оставаясь ограниченным, расходится и суммы гуляют к \(\pm \infty\)↩︎
