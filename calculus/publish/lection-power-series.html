Степенные ряды

Определение: Ряд вида \(\displaystyle \sum_{n = 0}^{\infty}a_n(x - x_0)^n\) - степенной ряд.

\(a_n\) - коэффиценты разложения,

\(a_0\) - свободный член,

\(x_0\) - центр разложения.

Замечание: Далее центр разложения \(0\).

Первая теорема Абеля

Теорема: (1-я теорема Абеля)

Рассмотрим ряд \(\displaystyle \sum_{n = 0}^{\infty}a_nx^n\).

  1. Если \(\exists x’ \ne 0 (= x_0): \,\,\,\, \displaystyle \sum_{n = 0}^{\infty}a_n\left(x’\right)^n\) сходится, то \(\forall x: \,\,\,\, |x| < |x’|\) ряд \(\displaystyle \sum_{n = 0}^{\infty}a_nx^n\) сходится абсолютно.
  2. Если \(\exists x’’ \ne 0 (= x_0): \,\,\,\, \displaystyle \sum_{n = 0}^{\infty}a_n\left(x’’\right)^n\) расходится, то \(\forall x: \,\,\,\, |x| > |x’’|\) ряд \(\displaystyle \sum_{n = 0}^{\infty}a_nx^n\) расходится.

Доказательство:

  1. Пусть \(|x| < |x’|\)

\(\displaystyle \sum_{n = 0}^{\infty}\left|a_nx^n\right| = \displaystyle \sum_{n = 0}^{\infty}\left|a_n\left(x’\right)^n\right|\left|\left(\frac{x}{x’}\right)^{n}\right|\)

\(\left|a_n\left(x’\right)^n\left(\frac{x}{x’}\right)^{n}\right| \leqslant M\cdot q^n, \,\,\,\, q = \left|\frac{x}{x’}\right| < 1\), \(M\) существует, так как член ряда стремится к нулю при \(n \to \infty\).

Получили бесконечно убывающую геометрическую прогрессию, а следовательно ряд сходится абсолютно \(\,\,\,\,\blacksquare\)

  1. Отрицание пункта \(1\).

Радиус сходимости

Определение: \([0, +\infty] \ni R = \sup\{x: \,\,\,\,\) ряд \(\displaystyle \sum_{n = 0}^{\infty}a_nx^n\) сходится \(\}\) - радиус сходимости ряда.

Теорема: (формула Коши-Адамара) Рассмотрим ряд \(\displaystyle \sum_{n = 0}^{\infty}a_nx^n\). Пусть \(\displaystyle \overline{\lim}_{n\to \infty} \sqrt[n]{|a_n|} = \frac1R \in [0, +\infty]\). Тогда \(R\) - радиус сходимости ряда \(\displaystyle \sum_{n = 0}^{\infty}a_nx^n\).

Доказательство: \(\forall x \in X \,\,\,\, \displaystyle \overline{\lim}_{n\to \infty} \sqrt[n]{|a_n||x|^n} = \frac{|x|}{R}\), а далее по признаку Коши \(\,\,\,\,\blacksquare\)

Свойства степенных рядов

Теорема: (свойства степенных рядов) Рассмотрим ряд \(\displaystyle \sum_{n = 0}^{\infty}a_nx^n\) и пусть \(R > 0\). Тогда

  1. \(\displaystyle \sum_{n = 0}^{\infty}a_nx^n \underset{[-R + \varepsilon, R - \varepsilon]}{\rightrightarrows} S(x) \,\,\,\, \forall \varepsilon > 0\),
  2. \(S(x) \in C(-R, R)\),
  3. \(\exists \displaystyle\int\limits_0^{x_0} a_nx^ndx = \displaystyle \sum_{n = 0}^{\infty}a_n\frac{x_0^{n + 1}}{n + 1}\),
  4. \(S(x) \in C^{\infty}(-R, R)\).

Доказательство:

  1. Ряд \(\displaystyle \sum_{n = 0}^{\infty}a_n\left(R - \varepsilon\right)^n\) сходится, а следовательно по признаку Вейерштрасса \(\displaystyle \sum_{n = 0}^{\infty}a_nx^n \underset{[-R + \varepsilon, R - \varepsilon]}{\rightrightarrows} S(x)\).
  2. Следует из пункта \(1\) и теоремы.
  3. Следует из пункта \(2\) и теоремы.
  4. Следует из \(3 \,\,\,\,\blacksquare\)
