Суммирование ряда Фурье

Определение: Пусть \(S_{n,\,f}(x)\) – это частичная сумма ряда Фурье функции \(f(x)\).

\(\sigma_{f,\,n}(x) = \frac1{n + 1} \displaystyle \sum_{k = 0}^{n}S_{k,\,f}(x)\) – последовательность Фейера функции \(f\).

\(F_{n}(x) = \frac1{n + 1} \displaystyle \sum_{k = 0}^{n}D_k(x)\) – ядро Фейера.

Утверждение: (свойства ядра Фейера)

  1. Ядро Фейера \(2\pi\)-периодическая, четная, непрерывная, неотрицательная,
  2. \(F_n(x) = \frac{\sin^2\left(\frac{n + 1}{2}x\right)}{2(n + 1)\sin^2\left(\frac{x}2\right)}\),
  3. \(\frac1{\pi}\displaystyle\int\limits_{-\pi}^{\pi} F_n(x)dx = 1\),
  4. \(\forall \delta \in (0, \pi) \,\,\,\, \displaystyle \lim_{n\to \infty}\displaystyle\int\limits_{\delta}^{\pi} F_n(x)dx = 0\).

Доказательство:

\(\,\,\,\,\blacksquare\)

Теорема: (Фейера) Пусть \(f(x) \in C(\mathbb{R})\) и \(f(x)\) – \(2\pi\)-периодическая функция. Тогда \(\sigma_{f, \,n}(x) \rightrightarrows f(x)\).

Доказательство:

\(S_{n,\, f}(x) = \frac{1}{\pi}\displaystyle\int\limits_{-\pi}^{\pi} f(\tau + x)D_n(\tau)d\tau \implies \sigma_{f, \, n}(x) = \frac{1}{\pi}\displaystyle\int\limits_{-\pi}^{\pi} f(\tau + x)F_n(\tau)d\tau\).

\[\left|\sigma_{f, \, n}(x) - f(x)\right| = \frac1{\pi}\left|\displaystyle\int\limits_{-\pi}^{\pi} f(\tau + x)F_n(\tau)d\tau - f(x)\displaystyle\int\limits_{-\pi}^{\pi} F_n(\tau)d\tau\right| = \frac1{\pi}\left|\displaystyle\int\limits_{-\pi}^{\pi} (f(\tau + x) - f(x))F_n(\tau)d\tau\right| \leqslant\]

\[\forall \varepsilon > 0 \,\,\,\, \exists \delta > 0: \,\,\,\, |f(t + x) - f(x)| < \varepsilon \,\,\,\, \forall |t| < \delta\]

\[M = \displaystyle\max_{\mathbb{R}}|f(x)|\]

\[\leqslant \frac1{\pi} \displaystyle\int\limits_{-\pi}^{-\delta} F_n(x)2Mdx + \frac1{\pi} \displaystyle\int\limits_{\delta}^{\pi} F_n(x)2Mdx + \frac1{\pi} \displaystyle\int\limits_{-\delta}^{\delta} (f(\tau + x) - f(x))F_n(\tau)d\tau \leqslant\] \[\leqslant \frac{2M}{\pi}\left(\displaystyle\int\limits_{-\pi}^{-\delta} F_n(x)dx + \displaystyle\int\limits_{\delta}^{\pi} F_n(x)dx\right) + \frac1{\pi} \displaystyle\int\limits_{-\delta}^{\delta} \varepsilon F_n(\tau)d\tau \leqslant \frac{2M}{\pi}\varepsilon + \frac{\varepsilon}{\pi}\]

Последнее верно, так как \(\displaystyle\int\limits_{\delta}^{\pi} F_n(x)dx = \displaystyle\int\limits_{\delta}^{\pi} \frac{\sin^2\left(\frac{n + 1}{2}x\right)}{2(n + 1)\sin^2\left(\frac{x}2\right)}dx \leqslant \displaystyle\int\limits_{\delta}^{\pi} \frac{1}{2(n + 1)\sin^2\left(\frac{x}2\right)}dx =\frac1{n + 1} \displaystyle\int\limits_{\delta}^{\pi} \frac{1}{2\sin^2\left(\frac{x}2\right)}dx\) и \(\forall \varepsilon > 0 \,\,\,\, \exists n \in \mathbb{N}: \,\,\,\, \frac1{n + 1} \displaystyle\int\limits_{\delta}^{\pi} \frac{1}{2\sin^2\left(\frac{x}2\right)}dx < \varepsilon \,\,\,\,\blacksquare\)
