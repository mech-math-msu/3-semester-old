Функциональные ряды

Определения

Определение: Пусть \(\{f_n(x)\}_{n = 1}^{\infty}\) определена на \(X \subset \mathbb{R}\). Пара \((f_n(x)\}_{n = 1}^{\infty}, \{S_N(x) = \displaystyle \sum_{n = 1}^{N}f_n(x)\}_{n = 1}^{\infty})\) - функциональный ряд на \(X\).

Обозначается: \(\displaystyle \sum_{n = 1}^{\infty}f_n(x)\)

Определение: (сходимость функционального ряда) Ряд \(\displaystyle \sum_{n = 1}^{\infty}f_n(x)\underset{X}{\rightarrow} S(x)\) 1, если \(\exists \displaystyle \lim_{n\to \infty}S_n(x) = S(x)\).

Определение: (равномерная сходимость функционального ряда) \(\displaystyle \sum_{n = 1}^{\infty}f_n(x) \underset{X}{\rightrightarrows}\), если \(S_n(x) \underset{X}{\rightrightarrows} S(x)\). 2

Критерии сходимости и прочие теоремы 3

Теорема: (критерий Коши сходимости функционального ряда) \(\displaystyle \sum_{n = 1}^{\infty}f_n(x)\underset{X}{\rightrightarrows} S(x) \iff \forall \varepsilon > 0 \,\,\,\, \exists N \in \mathbb{N}: \,\,\,\,\forall x \in X, \,\,\,\, \forall k, m > N \,\,\,\, \left|\displaystyle \sum_{n = k + 1}^{m}f_n(x)\right| < \varepsilon\).

Доказательство: см. определение функционального ряда 4 \(\,\,\,\,\blacksquare\)

Теорема: (необходимое условие равномерной сходимости функционального ряда) \(\displaystyle \sum_{n = 1}^{\infty}f_n(x)\underset{X}{\rightrightarrows}S(x) \implies \left|f_n(x)\right|\underset{X}{\rightrightarrows} 0\).

Доказательство: Критерий Коши при \(m = k \,\,\,\,\blacksquare\)

Теорема: (признак Вейерштрасса равномерной сходимости функционального ряда) Рассмотрим \(\displaystyle \sum_{n = 1}^{\infty}f_n(x)\). Если \(\exists \{c_n\}_{n = 1}^{\infty}\), такая что \(\displaystyle \sum_{n = 1}^{\infty}c_n\) сходится и \(\forall n \in \mathbb{N} \,\,\,\, \forall x \in X \,\,\,\, |f_n(x)| < c_n\), то \(\displaystyle \sum_{n = 1}^{\infty}f_n(x)\underset{X}{\rightrightarrows}\).

Доказательство: \[\left|\displaystyle \sum_{n = k + 1}^{m}f_n(x)\right| \leqslant \displaystyle \sum_{n = k + 1}^{m}\left|f_n(x)\right| \leqslant \displaystyle \sum_{n = k + 1}^{m}\left|c_n\right| < \varepsilon \,\,\,\,\blacksquare\]

Теорема: (признак Дирихле)

Ряд \(\displaystyle \sum_{n = 1}^{\infty}f_n(x)g_n(x)\underset{X}{\rightrightarrows}\), если

  1. \(\exists M > 0: \,\,\,\, \forall n \in \mathbb{N} \,\,\,\, \forall x \in X \,\,\,\, \left|f_n(x)\right| < M\),
  2. \(\displaystyle \lim_{n\to \infty}g_n(x) = 0, \,\,\,\, \forall x \in X \,\,\,\, \{f_n(x)\}_{n = 1}^{\infty}\) монотонна.

Доказательство: аналогично признаку Дирихле \(\,\,\,\,\blacksquare\)

Теорема: (признак Абеля)

Ряд \(\displaystyle \sum_{n = 1}^{\infty}f_n(x)g_n(x)\underset{X}{\rightrightarrows}\), если

  1. \(\displaystyle \sum_{n = 1}^{\infty}f_n(x)\underset{X}{\rightrightarrows}\),
  2. \(\forall x \in X\) последовательность \(\{g_n(x)\}_{n = 1}^{\infty}\) монотонна и ограничена.

Доказательство: аналогично признаку Абеля \(\,\,\,\,\blacksquare\)

  1. Сходится к \(S(x)\)↩︎

  2. Функциональный ряд \(\displaystyle \sum_{n = 1}^{\infty}f_n(x)\) равномерно сходится к \(S(x)\), если функциональная последовательность его частичных сумм равномерно сходится к \(S(x)\).↩︎

  3. Все просто переформулировка теорем для последовательностей.↩︎
