Эйлеровы интегралы

Теорема: Пусть \(\Gamma_n(x) = \frac{(n - 1)!\cdot n^x}{\displaystyle \prod_{k = 0}^{n - 1}\left(x + k\right)}, \,\,\,\, x \not\in \{0, -1, -2, \ldots\}\).

\[\exists \displaystyle \lim_{n\to \infty}\Gamma_n(x) = \Gamma(x)\]

Доказательство:

\[\Gamma_n(x) = \frac{(n - 1)!\cdot n^x}{\displaystyle \prod_{k = 0}^{n - 1}\left(x + k\right)} = \frac{1}{x \displaystyle \prod_{k = 1}^{n - 1}\left(\frac{x}{k} + 1\right)}\cdot \frac{n^x}{(n - 1)^x}\cdot \ldots \cdot \frac{2^x}{1^x} = \frac{1}{x} \displaystyle \prod_{i = 1}^{n - 1}\left(\frac{x}{k} + 1\right)^{-1}\cdot \left(\frac{1}{k} + 1\right)^x \,\,\,\,\blacksquare\]

Определение: \(\Gamma(x)\) – гамма функция Эйлера.
